Arquitectura MCP: La fuente de verdad

Model Context Protocol como puente entre el Design System real y la generacion de prototipos

🤖 Claude Code + Skills RPP Orquesta la generacion, aplica reglas anti-alucinacion, ejecuta specs
🔌 MCP Server (@dropi/ui-mcp) Extrae specs de Storybook + design tokens. 69 componentes indexados
🎨 DESIGN_SYSTEM_RULES.md (457 lineas) Tokens, spacing, breakpoints, colores, tipografia — la ley del DS
🛠 core_front (Angular 18 + Nx) 69 componentes UI en src/app/ui/ + Storybook → ui.dropi.co

MCP = Source of Truth. El servidor MCP extrae automaticamente las specs de cada componente desde Storybook y los design tokens. Claude Code consulta estas specs en tiempo real antes de generar cualquier linea de codigo.

Clean Code desde el origen
